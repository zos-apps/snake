## Features

- Classic gameplay
- Score tracking
- Speed increase
- High scores

## Installation

### From App Store
Open the **App Store** in zOS and search for "Snake".

### Via npm
```bash
npm install @zos-apps/snake
```

## Usage

- Arrow keys to move
- Eat food to grow
- Avoid walls and yourself
